2. Redact/blur: user name, avatar, email, school, any real assignment text, and any IDs in URLs.
3. Crop out admin/debug panels and browser autofill.
4. Confirm no query strings, tokens, or project identifiers are visible.
5. Export at a consistent width; keep file names as above.
